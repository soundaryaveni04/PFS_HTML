# PFS_HTML

A collection of HTML and CSS projects for learning web development.

## Projects

### 1. **Cardiac Diet Information Page** (`helthcare.html`)
An educational HTML page about cardiac diet that includes:
- Information about cardiovascular health
- List of recommended foods (fish, lean meats, low-fat dairy)
- Multiple list formats (unordered, ordered, and custom styling)
- Yellow background styling

### 2. **Smile Face Animation** (`sm.html` + `sm.css`)
A creative CSS and HTML project that creates a smiley face with:
- Heart-shaped ears
- Eyes with pupils
- Animated lips/smile
- Styled using custom CSS

## Files Included

- `helthcare.html` - Cardiac diet educational page
- `sm.html` - Smiley face HTML structure
- `sm.css` - Styling for the smiley face animation
- `bulb` - Additional project file

## Purpose

These are practice projects demonstrating HTML structure, CSS styling, and basic animations for learning web development fundamentals.
